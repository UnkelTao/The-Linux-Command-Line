\chapter{文件系统中跳转}

我们需要学习的第一件事（除了打字之外）是如何在 Linux 文件系统中跳转。 在这一章节中，我们将介绍以下命令：
\begin{itemize}
	\item pwd — 打印出当前工作目录名
	\item cd — 更改目录
	\item ls — 列出目录内容
\end{itemize}
